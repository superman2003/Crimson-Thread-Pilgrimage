# 六章丝之歌级大体量地图蓝图规格

## 目标

- 每章按 55-75 个房间设计，强调长主线、密支线、能力回访、捷径闭环和 Boss 前短跑。
- 本规格不复刻任何商业地图，只抽象大体量银河恶魔城的结构密度与节奏。
- 高清 PNG 用于外部软件复绘；JSON 用于后续 Godot/Tiled/Tres 关卡数据落地。

## 公共图例

- 红箭头：一周目主线推进。
- 蓝线：支线、奖励房、区域内连接。
- 绿虚箭：从背面开启的捷径。
- 紫虚箭：能力门或回访门，图中标注所需能力。
- S/N/R/M/A/BOSS/IN/OUT：存档、NPC、奖励、精英战、能力、Boss、章节入口、章节出口。

## 章节规格

### 第一章 苔铃外庭

- 主题：苔藓庭院、钟廊、湿木台阶、地下根道
- 体量：64 房，10 个区域簇，主线 35 房。
- 点位：捷径 8，能力门 5，存档 7，NPC 6，奖励 9，精英战 2。
- 能力：丝线钩刺 / 下劈反弹 / 苔铃钥记
- Boss：苔铃主母
- 回环：用钟塔纵井把前庭、温室、巢厅三层打通，Boss 前存档到战场不超过 2 房间。
- 蓝图：`ch01_moss_bell_court_blueprint.json`
- 图片：`ch01_moss_bell_court_blueprint.png`

### 第二章 铸雨渠

- 主题：雨水工厂、齿轮泵、铁渠、蒸汽阀门
- 体量：68 房，10 个区域簇，主线 39 房。
- 点位：捷径 8，能力门 5，存档 7，NPC 6，奖励 9，精英战 2。
- 能力：阀门冲刺 / 水轮二段跳 / 赤锈耐热
- Boss：绯线剪刀使徒
- 回环：核心是横向长渠加两根竖井，阀门捷径让后半段回到主渠中点。
- 蓝图：`ch02_rain_foundry_canal_blueprint.json`
- 图片：`ch02_rain_foundry_canal_blueprint.png`

### 第三章 盐白档案馆

### 第六章 无声冠核

## 验收阈值

- 每章房间数 >= 55。
- 每章区域簇 >= 8。
- 每章捷径 >= 8。
- 每章能力门 >= 6。
- 每章存档点 >= 5，且 Boss 前 3 房间内有存档点。
- 每章至少 5 个 NPC、8 个奖励点、2 个精英战。
- 每张 PNG 宽 >= 4096、高 >= 2500，且包含图例、编号、主线、支线、捷径、能力门、Boss/NPC/存档/奖励标记。
