# 《绯线远征》6章 Route Graph 路径规划

生成时间：2026-06-15 16:11:47.827  
负责 Agent：Agent-Route  
范围：只读关卡路线建议，不改源码。本文用于把现有章节制原型升级为更接近魂游箱庭的中型闭环设计。

## 0. 总体路由原则

目标不是把 6 章做成单向横版关卡，而是让每章像一座小型可记忆空间：玩家先看见未解锁路、被迫绕行、打开单向捷径、回到篝火/安全屋，再以新能力返场拿奖励。

通用节点规则：

| 类型 | 记号 | 用途 |
|---|---|---|
| 入口/主干 | `N00` 起 | 首次推进路径，必须自然读懂本章主题 |
| 篝火/灯碑 | `BF` | 复活、补给、章节传送点，Boss 前必须有一个 |
| 安全屋 | `SH` | 无敌人或低威胁房间，用于观察机关、回路汇合、NPC/文本 |
| 能力门 | `G` | 需要章节能力或后续能力返场开启 |
| 捷径 | `SC` | 从深处回连早期区域，优先做单向开启 |
| 支线 | `S` | 非必经，奖励素材、武器、护符、剧情碎片 |
| 精英/Boss | `E/B` | 验证本章机制掌握度 |

全局能力返场顺序：

| 获取点 | 能力/物品 | 立即用途 | 返场用途 |
|---|---|---|---|
| 第 1 章 Boss 后 | 裂隙钩索 | 对空、越过断层、拉近射手 | 打开第 1/2/4/5 章高位支路 |
| 第 2 章 Boss 后 | 二段跳 | 空战续航、镜台高度容错 | 打开第 1/3/5 章高处回路 |
| 第 3 章精英/主线 | 炉心重锤/空中下劈弹跳 | 破盾、弹跳穿越热区 | 打开第 1/3/6 章重门和地板裂缝 |
| 第 4 章 Boss 后 | 斜向冲刺 | 抢潮汐窗口、空中追击 | 打开第 2/4/5/6 章远距斜线门 |
| 第 5 章 Boss 后 | 反闪强化/星线弩刃 | 回耐、远距安全输出 | 打开第 6 章最终试炼变体和高危返场奖励 |
| 第 6 章通关 | 二周目挑战 | 高节奏复盘 | 所有章节开放红线变体门 |

推荐全局体验顺序：第 1 章线性教学但有可见锁门 -> 第 2 章第一次强支线预览 -> 第 3 章用安全格学习机关节拍 -> 第 4 章双路线分叉 -> 第 5 章垂直箱庭 -> 第 6 章回声式总考试。每章首次通关后，都要至少留下 2 个“现在还不能拿”的可见目标，保证返场动机。

## 1. 第 1 章：翠锈钟庭 `moss-forge`

核心体验：基础跳跃、短冲刺、读招、周期机关。目标是让玩家第一次理解“前进不是一直向右”，而是围绕一座钟庭开回路。

### Route Graph

```text
C1-N00 入口苔钟门廊
  -> C1-BF1 入门灯碑
  -> C1-N01 翠锈齿廊
  -> C1-N02 摆钟训练廊
  -> C1-S01 上层花钟竖井支线 [可见，需要二段跳/钩索返场拿全]
  -> C1-SH1 齿轮检修安全屋
  -> C1-N03 铜叶庭院锁门战
  -> C1-SC1 断钟机房单向门 [从 N04 回开到 N01]
  -> C1-N04 断钟机房
  -> C1-E01 断齿钟卫
  -> C1-BF2 Boss 前灯碑
  -> C1-B01 锈冠守卫
  -> C1-EXIT 裂隙钩索获得

返场边：
C1-N01 --裂隙钩索--> C1-S02 高钟索道素材台
C1-N03 --炉心重锤--> C1-S03 锈封钟核门
C1-N04 --二段跳--> C1-S01 花钟竖井顶层
```

### 关键节点编号

| 节点 | 类型 | 内容 | 奖励/门槛 |
|---|---|---|---|
| `C1-BF1` | 篝火 | 入口灯碑，学习恢复与死亡回点 | 基础补给 |
| `C1-N02` | 主线 | 摆钟锤、齿轮平台、低伤雾障 | 教会周期机关 |
| `C1-S01` | 支线 | 花钟竖井，顶端先看见素材但够不到 | 初次可拿星屑，二段跳返场拿护符碎片 |
| `C1-SH1` | 安全屋 | 齿轮检修间，可从窗口看见 Boss 塔影 | 世界观文本、短休息 |
| `C1-SC1` | 捷径 | 从断钟机房背面打开，直通翠锈齿廊 | 死亡后少走 60% 路程 |
| `C1-E01` | 精英 | 断齿钟卫，测试绕背和重击 | `rust_guard_core` |
| `C1-BF2` | 篝火 | Boss 前灯碑 | 距 Boss 门 8 秒内 |

### 推荐玩家体验顺序

1. 从 `BF1` 学基础跳跃和短冲刺，看到 `S01` 顶部发光物但拿不到。
2. 经 `N02` 到 `SH1`，让玩家在安全窗观察摆钟和齿轮循环。
3. 进入 `N03` 锁门战，战后从高台看见 `SC1` 背面。
4. 通过 `N04` 打开 `SC1`，回到 `N01`，形成第一次“深处回家”的箱庭感。
5. 从捷径再推到 `E01` 与 `BF2`，挑战 Boss。

### 捷径解锁顺序

`SC1` 必须在 `N04` 后半段回开；`SC2` 可在 Boss 后用裂隙钩索从 `BF2` 上方拉到 `N01` 高层，作为返场快速线；`SC3` 二周目在 `S03` 锈封钟核门后打开，连接入口和隐藏精英。

### 回篝火/安全屋路线

死亡常规回 `BF1`；打开 `SC1` 后，从 `BF1 -> N01 -> SC1 -> N04` 可 25 秒内回到精英前；Boss 失败回 `BF2`。`SH1` 是章节中段安全屋，不复活但可放商人/文本/补给。

### 探索奖励

首通：裂隙钩索、`rust_crown_core`。  
支线：花钟竖井星屑、锈封钟核门内的耐力碎片、高钟索道的素材包。  
返场：二段跳拿 `S01` 顶部护符“钟摆余响”；炉心重锤开 `S03` 得隐藏武器强化素材。

### 二周目/返场路径

二周目从 `BF1` 可直接看到红线钟影门，需反闪强化进入；路线为 `BF1 -> S03 -> 隐藏精英 -> SC3 -> BF2`。返场推荐在第 2 章拿二段跳后回 `S01`，第 3 章拿重锤后回 `S03`。

## 2. 第 2 章：镜雨档案馆 `glass-rain`

核心体验：钩索、延迟影像、真假空间。箱庭重点是“看见镜中路，但先从真实空间绕到背面”。

### Route Graph

```text
C2-N00 雨廊阅览台
  -> C2-BF1 入馆灯碑
  -> C2-N01 反页书库
  -> C2-G01 高位钩索井 [需要第 1 章裂隙钩索]
  -> C2-S01 镜柜支线A [击碎镜柜开启]
  -> C2-SH1 静音阅览井
  -> C2-N02 倒影索桥
  -> C2-SC1 镜柜背门 [回连 N01]
  -> C2-N03 封存档案室
  -> C2-E01 复写装订官
  -> C2-BF2 Boss 前观测灯碑
  -> C2-B01 镜雨司书
  -> C2-EXIT 二段跳获得

返场边：
C2-N01 --斜向冲刺--> C2-S02 破窗雨廊
C2-N02 --二段跳+钩索--> C2-S03 倒影上层索桥
C2-N03 --反闪强化--> C2-S04 白书脊红线门
```

### 关键节点编号

| 节点 | 类型 | 内容 | 奖励/门槛 |
|---|---|---|---|
| `C2-BF1` | 篝火 | 入馆灯碑，靠近第一个可碎镜柜 | 返场传送点 |
| `C2-G01` | 能力门 | 钩索井，要求拉住折光锚点 | 首通必经能力门 |
| `C2-S01` | 支线 | 镜柜支线A，假路径提示真路径位置 | 镜片素材 |
| `C2-SH1` | 安全屋 | 静音阅览井，无敌人，能看见上下两条影子路 | 机关预读 |
| `C2-SC1` | 捷径 | 镜柜背门，打开后从 `N03` 回 `N01` | 快速回中段 |
| `C2-E01` | 精英 | 复写装订官，真假身测试 | `mirror_binding` |
| `C2-BF2` | 篝火 | Boss 前观测灯碑 | Boss 重试 |

### 推荐玩家体验顺序

1. `BF1` 后立刻放一处镜中宝箱，让玩家知道这里有“看得见够不着”。
2. `G01` 要求使用裂隙钩索，确认第 1 章能力不是只服务战斗。
3. 通过 `S01` 镜柜短支路得到第一个小奖励，同时从镜中看见 `SC1` 背门。
4. 在 `SH1` 观察延迟镜台，再进入 `N02` 的实战索桥。
5. 打开 `SC1` 后，玩家从 `N01` 可直接去 `N03`，形成档案馆回环。

### 捷径解锁顺序

先碎 `S01` 镜柜得到侧路，再在 `N03` 背面打开 `SC1`；Boss 后获得二段跳，立刻让 `BF2` 上方出现可跳回 `N02` 上层的返场路线；第 4 章拿斜向冲刺后解 `S02` 破窗雨廊。

### 回篝火/安全屋路线

首次中段死亡回 `BF1`，但 `SC1` 开后可从 `BF1 -> N01 -> SC1 -> N03` 快速回精英前。`SH1` 安排在 `N01/N02` 之间，是镜台教学和压力重置点。Boss 失败回 `BF2`，Boss 门到战斗起点不超过 10 秒。

### 探索奖励

首通：二段跳、`mirror_index_core`。  
支线：镜柜碎片、折光书签、盐镜双刃素材。  
返场：二段跳+钩索拿 `S03` 上层护符“迟影书脊”；斜冲拿 `S02` 破窗雨廊的星核上限碎片。

### 二周目/返场路径

二周目红线变体让镜像敌人保留玩家上一次攻击方向。推荐路线为 `BF1 -> G01 -> S03 -> E01 变体 -> BF2`。返场优先级：第 2 章通关后立即用二段跳回 `S03`，第 4 章后回 `S02`。

## 3. 第 3 章：余烬织炉 `ember-loom`

核心体验：热浪窗口、升降炉架、破甲。箱庭重点是多条热区短环，安全格不是休息室，而是观察下一段节拍的阳台。

### Route Graph

```text
C3-N00 熄炭入口
  -> C3-BF1 炉前灯碑
  -> C3-N01 织线炉架
  -> C3-SH1 冷却安全格A
  -> C3-S01 熔线侧窑 [需要二段跳拿全]
  -> C3-N02 红线绞盘井
  -> C3-SC1 冷却闸门 [从 N02 回连 N01]
  -> C3-N03 锻缝车间
  -> C3-E01 熔线监工 [掉炉心钥/重锤素材]
  -> C3-SH2 冷却安全格B
  -> C3-N04 炉心导管
  -> C3-BF2 Boss 前炉灯
  -> C3-B01 焰织屠夫
  -> C3-EXIT 空中下劈弹跳稳定化

返场边：
C3-N01 --炉心重锤--> C3-S02 裂炉底仓
C3-N02 --斜向冲刺--> C3-S03 快速导管上层
C3-N04 --反闪强化--> C3-S04 熔线红门
```

### 关键节点编号

| 节点 | 类型 | 内容 | 奖励/门槛 |
|---|---|---|---|
| `C3-BF1` | 篝火 | 炉前灯碑，靠近第一段热浪教学 | 补给 |
| `C3-SH1` | 安全屋 | 冷却安全格A，可观察熔线切割周期 | 无战斗 |
| `C3-S01` | 支线 | 熔线侧窑，首通可拿低层素材，高层需二段跳 | 火线素材 |
| `C3-SC1` | 捷径 | 冷却闸门，从绞盘井背面打开 | 回连 `N01` |
| `C3-E01` | 精英 | 熔线监工，破甲/打断训练 | 炉心重锤素材或钥 |
| `C3-SH2` | 安全屋 | Boss 前第二个冷却安全格 | 观察高速机关 |
| `C3-BF2` | 篝火 | Boss 前炉灯 | 快速重试 |

### 推荐玩家体验顺序

1. `N01` 用低风险热浪教学“红线预警后行动”。
2. `SH1` 给玩家看见 `S01` 高处奖励，提示二段跳返场。
3. `N02` 竖井中先下后上，打开 `SC1` 后回到 `N01`，降低死亡惩罚。
4. `N03` 锁门战加入炉甲突击者，让玩家理解破甲需求。
5. `E01` 掉落关键素材，之后 `SH2 -> BF2 -> Boss`。

### 捷径解锁顺序

`SC1` 从 `N02` 中段打开，是第一个主捷径；`SC2` 在拿炉心重锤后从 `N01` 打裂地板进 `S02`，可通到 `N03` 背面；`SC3` 斜向冲刺后从 `S03` 快速到 `BF2`。

### 回篝火/安全屋路线

首通死亡回 `BF1`；`SC1` 开后 `BF1 -> N01 -> SC1 -> N02` 可快速回竖井。`SH1` 和 `SH2` 分别服务机关教学和 Boss 前降压。Boss 失败回 `BF2`，路径上不再放敌人。

### 探索奖励

首通：空中下劈弹跳、`emberloom_heart`。  
支线：熔线侧窑的火线素材、裂炉底仓的护符“余温踏点”、快速导管上层的武器强化矿。  
返场：二段跳补全 `S01`，炉心重锤开 `S02`，斜向冲刺开 `S03`。

### 二周目/返场路径

二周目让热浪间隔缩短，红线预警减少 0.2 秒。推荐路线 `BF1 -> S02 -> N03 -> E01 变体 -> S03 -> BF2`。返场优先在第 4 章拿斜向冲刺后回来开 `S03`。

## 4. 第 4 章：盐月水渠 `moon-salt`

核心体验：水位、流速、窄平台、路线选择。箱庭重点是上层安全慢路和下层危险快路在多个节点交叉。

### Route Graph

```text
C4-N00 月盐闸口
  -> C4-BF1 闸口灯碑
  -> C4-N01 白潮暗渠
  -> C4-S01 上层干渠支线 [安全慢路]
  -> C4-N02 逆流泵井
  -> C4-SH1 闸门观测室
  -> C4-G01 下层满水渠 [需要二段跳+下劈/可首通低配通过]
  -> C4-N03 盐蚀分流室
  -> C4-SC1 月闸回廊 [回连 N01]
  -> C4-N04 月池净化间
  -> C4-E01 盐月司潮者
  -> C4-BF2 Boss 前净水灯碑
  -> C4-B01 盐月歌姬
  -> C4-EXIT 斜向冲刺获得

返场边：
C4-N02 --裂隙钩索--> C4-S02 高位水车台
C4-N03 --斜向冲刺--> C4-S03 银潮极速线
C4-N04 --反闪强化--> C4-S04 月盐红线闸
```

### 关键节点编号

| 节点 | 类型 | 内容 | 奖励/门槛 |
|---|---|---|---|
| `C4-BF1` | 篝火 | 闸口灯碑 | 章节起点 |
| `C4-S01` | 支线/慢路 | 上层干渠，敌人少但绕远 | 安全玩家路线 |
| `C4-G01` | 能力门/快路 | 下层满水渠，水流密、奖励多 | 高风险快线 |
| `C4-SH1` | 安全屋 | 闸门观测室，可看水位三档变化 | 机关预读 |
| `C4-SC1` | 捷径 | 月闸回廊，从 `N03` 回连 `N01` | 中段回路 |
| `C4-E01` | 精英 | 盐月司潮者，节拍打断训练 | `conductor_moon_chime` |
| `C4-BF2` | 篝火 | Boss 前净水灯碑 | 重试点 |

### 推荐玩家体验顺序

1. `N01` 先给一段低水位安全平台，解释水位改变可通行区域。
2. `S01` 和 `G01` 在同屏或邻屏分叉：上层安全、下层危险且有奖励。
3. 玩家无论走哪条路都汇入 `N03`，并从背面打开 `SC1`。
4. `SH1` 设置在分流前，让玩家看到水位循环再选择路线。
5. `N04/E01` 强化节拍和流速，Boss 前回到 `BF2`。

### 捷径解锁顺序

`SC1` 从 `N03` 打开，是主回路；`SC2` 用裂隙钩索从高位水车台直达 `N04` 侧门；`SC3` 拿斜向冲刺后从 `S03` 快速穿过银潮极速线到 `BF2`。

### 回篝火/安全屋路线

初期死亡回 `BF1`；`SC1` 开后 `BF1 -> N01 -> SC1 -> N03` 可快速回分流室。`SH1` 作为观察室和路线选择前哨。Boss 失败回 `BF2`，Boss 门前不放水位惩罚。

### 探索奖励

首通：斜向冲刺、`saltmoon_voice_core`。  
支线：上层干渠的稳定护符、下层满水渠的高价值星屑、水车台的钩索素材。  
返场：斜向冲刺打开 `S03` 银潮极速线，反闪强化打开 `S04` 红线闸。

### 二周目/返场路径

二周目水位变化更频繁，下层满水渠奖励升级。推荐路线 `BF1 -> G01 下层快线 -> S03 -> E01 变体 -> BF2`。返场优先用斜向冲刺练极速线，用反闪强化解红线闸。

## 5. 第 5 章：骨枝空园 `bone-orchard`

核心体验：垂直探索、虚实平台、风压空战。箱庭重点是从底层仰望顶层，再用多条下落门把高层回接入口。

### Route Graph

```text
C5-N00 空园残桥
  -> C5-BF1 底层灯碑
  -> C5-N01 骨枝温室
  -> C5-S01 左翼温室支线 [需要钩索/二段跳]
  -> C5-SH1 风铃温室安全屋
  -> C5-N02 无根天井
  -> C5-SC1 下落门A [从天井中层回到底层]
  -> C5-N03 枯冠回廊
  -> C5-SC2 下落门B [从回廊回 BF1 附近]
```

## 8. 实现落地建议

1. 灰盒优先做节点和回路，不先铺完整美术。每章至少落 `BF1`、`SH1`、`SC1`、`BF2` 四个箱庭锚点。
2. 每个能力门要有“正向必用一次”和“返场奖励一次”。否则能力会像横版技能，而不是探索钥匙。
3. 每个 Boss 前篝火必须通过一个已解锁捷径或安全屋回路抵达，避免魂游感只剩惩罚。
4. 章节制入口可以保留，但章节内部不要只用 `spawn -> exit`，后续数据结构可新增 `nodes`、`edges`、`locks`、`shortcuts`、`rewards`。

## 10. 风险与回滚

风险：

| 风险 | 影响 | 应对 |
|---|---|---|
| 现有原型是横向 `spawn -> exit` 数据 | 无法直接表达回环 | 先把本文当灰盒规格，后续再扩展地图节点数据 |
| 6 章一次做箱庭过大 | 关卡制作周期膨胀 | 先完整实现第 1 章闭环，再复制节点模板 |
| 能力门过多导致首通卡死 | 玩家路径不清 | 首通主线只强制 1 个能力门，其余放返场支线 |
| Boss 前跑路过长 | 挫败感升高 | 每章 `BF2` 到 Boss 控制在 8-10 秒 |

回滚：

1. 若实现周期不足，只保留每章 `BF1 -> 主线 -> SC1 -> BF2 -> Boss` 的最小闭环。
2. 若能力门实现不足，先把返场支线改为普通宝箱门，但保留节点编号。
3. 若终章枢纽超范围，先按 `R1-R5` 线性串联实现，但每完成一个回声室仍回到 `HUB`。
