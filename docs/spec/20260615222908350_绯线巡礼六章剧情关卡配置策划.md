## 0. 配置化字段约定

本文每章都按同一组字段组织，后续可拆为 JSON：

- `chapter_id`：章节唯一 ID，建议 `ch01_moss_bell_court` 这类蛇形命名。
- `name`：中文章节名。
- `area_theme`：区域主题，包含视觉、机关、叙事关键词。
- `main_goal`：主线目标，进入章节时显示在地图/任务 UI。
- `story_beats`：关键故事推进，按进入、发现、反转、Boss 后四段写。
- `rooms`：可转成 `map_rooms` 的区域节点，包含 `objective`、`guide`、`danger`、`next`。
- `npcs`：至少 3 个 NPC，包含 `id`、`name`、`role`、`position_hint`、`dialogue`。
- `enemy_points`：至少 5 个敌人点，包含 `id`、`room`、`spawn_mix`、`combat_role`、`story_use`。
- `boss`：本章 Boss 的剧情身份、招式主题、掉落与后续门禁。
- `gates`：能力门/钥匙门，和 `progression_plan.json` 的 `gate_types` 对齐。
- `key_collectibles`：主线收集、地图收集、升级材料、结局线索。
- `ending_clues`：本章提供给三结局的证据。
- `asset_adaptation`：免费/开源/已导入授权素材适配建议，优先使用 CC0、Public Domain、MIT 或项目已记录免费素材。

全局故事口径：旧王朝用巨钟封住深井黑潮，却把王冠钉在钟舌上。六座失声之城被锈钟、铸雨、盐晶、断弦、黑曜与王冠核心隔开。主角是能看见绯色命线的巡礼者，目标是收集六枚失声印记，并在终章选择重铸钟声、切断封印，或用真相改写王冠契约。

## 2. CH01 苔钟庭

`chapter_id`: `ch01_moss_bell_court`  
`name`: 第一章：苔钟庭  
`source_alignment`: 对齐当前 `demo_ch01_moss_bell_court.json` 的试玩章结构：7 个房间、3 把钟钥、5 个 NPC、5 个敌人点、Boss 锈冠守卫。  
`area_theme`: 苔藓覆盖的旧钟庭，青绿色苔石、黄铜桥、假苔地、落钟舌与锈冠殿。  
`main_goal`: 收集 3 把钟钥，从后门打开捷径，击败锈冠守卫，取得第一枚失声印记。

### story_beats

1. 进入：织线学徒教会左键斩击、右键缝补、E 绯线爆裂、F 交互，强调“苔钟庭只是第一段巡礼”。
2. 发现：守门朝圣者说明正门被锈钟封死，必须绕行外庭、升台、钟廊找三把钟钥。
3. 反转：后门司铃人透露王冠不是头饰，而是倒扣在城上的巨钟；玩家打开捷径后看见王冠回廊。
4. Boss 后：锈冠守卫体内掉出温热的王冠裂片，第一枚失声印记发出不属于苔钟庭的水声，引向铸雨渠。

## 3. CH02 铸雨渠

`chapter_id`: `ch02_rain_foundry_canal`  
`name`: 第二章：铸雨渠  
`area_theme`: 下水铸坊、雨声管网、水轮升降、热铁冷却雾。  
`main_goal`: 沿雨渠进入旧铸坊，找到被水声藏起的铁匠印记，解锁墙面攀附与回声钩索。

### story_beats

1. 进入：第一章印记指向一条一直下雨的地下水渠，水声会盖住 NPC 的关键字。

4. Boss 后：沉锤铁匠把自己的钩环交给主角，要求主角去盐白书库找契约原文。

## 10. 风险与回滚

- 风险：第二章后每章同时给两个能力会增加实现成本。回滚：先保留主线必需能力，另一个能力作为 Boss 后升级或支线奖励延后。
- 风险：终章三结局需要额外状态记录。回滚：先用 `story_flags` 布尔值记录五章证词，UI 暂时只显示结局文本。
- 风险：免费素材授权口径混杂。回滚：运行时优先保留 CC0/Public Domain/MIT/Apache-2.0/已记录免费素材；itch 免费包仅友方 NPC 静态占位，敌人/Boss 禁用。
- 风险：当前 demo 是单章横向场景，后续 6 章箱庭规模较大。回滚：每章先实现 `入口 -> 中段捷径 -> Boss 前灯 -> Boss -> 下一章入口` 的最小闭环，再加支线和返场门。
