# 技能道具门禁数据契约

时间：2026-06-15 20:20:03

## 文件

`godot/data/progression_plan.json`

## 内容

- `abilities`：技能表，包含解锁章节、输入、类型、说明和解锁门禁。
- `items`：关键道具、货币、升级材料。
- `charms`：护符系统第一版，包含消耗槽位和效果。
- `gate_types`：能力门类型，每种门声明需要哪些能力。
- `target_scale`：6 章、约 126 房间的目标规模。

## 运行时接入建议

后续新增 `progression_state.gd`，负责读取技能和道具状态。玩家脚本、地图门、机关和 UI 不直接硬编码技能名，而是通过 `has_ability(id)`、`has_item(id)`、`can_open_gate(gate_type)` 查询。
