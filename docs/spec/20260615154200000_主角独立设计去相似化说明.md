# 主角独立设计去相似化说明

## 结论
旧版主角存在白面具、角状轮廓、红披风、细针武器等高相似风险，后续不建议作为主用角色。

## 新主角
- 名称：灯芯织者 Lumen Spoolwright
- 核心身份：携带线轮与灯芯装置的流浪工匠/冒险者
- 头部：发光玻璃灯笼结构，不使用白面具、虫形轮廓或黑色空洞眼
- 服装：深靛工装、短外套、缠布靴、黄铜织机背架
- 武器：黄铜线轮环刃与折叠剪，不使用针剑或细剑
- 视觉识别：暖琥珀灯光、黄铜机构、青绿色发光线、工匠装置感

## 明确规避
- 不使用虫、蛾、蜂、螳螂等昆虫拟人特征
- 不使用角、触角、白色面具、黑椭圆眼
- 不使用红披风骑士轮廓
- 不使用针剑作为主武器
- 不使用现有同类游戏角色的标志性比例与姿态
