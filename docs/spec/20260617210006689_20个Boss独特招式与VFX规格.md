# 20 个 Boss 独特招式与 VFX 规格

## 目标

每个 Boss 拥有 2-3 个独特攻击方式，并为每个攻击提供对应 VFX、6 帧关键帧节奏和生成关键词。素材导入时保留旧 `attack` 默认动作，并新增 `attacks[]` / `special_attacks` 多招式结构。

## 统一素材规范

- 每个特殊攻击 6 帧：ready、anticipation、windup、active_hit、follow_through、recovery。
- 每个 Boss 最终至少 2 个特殊攻击，推荐 3 个。
- 每个攻击必须包含独立 VFX：`key`、中文名、英文名、主色、触发帧。
- 原始源条带保存到 `artifacts/boss_keyframes/<boss>/source/<attack_id>/generated_attack_strip_chroma.png`。
- Artifact 帧输出到 `artifacts/boss_keyframes/<boss>/actions/<attack_id>/attack_00.png` 到 `attack_05.png`。
- Godot 帧输出到 `godot/assets/sprites/bosses/<boss>/frames/<attack_id>/attack_00.png` 到 `attack_05.png`。
