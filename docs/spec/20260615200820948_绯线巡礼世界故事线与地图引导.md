# 绯线巡礼世界故事线与地图引导

时间：2026-06-15 20:08:20

## 世界线

游戏名：绯线巡礼

核心设定：旧王朝用巨钟封住深井里的黑潮，却把王冠也钉在钟舌上。多年后，所有钟声变成铁锈，城邦道路被怪物和断裂机关隔开。主角是能看见绯色命线的巡礼者，要收集六枚失声印记，查明是谁让钟声沉默。

## 六章规划

1. 苔钟庭：学习地图、捷径、钟钥、近战怪前摇，击败锈冠守卫。
2. 铸雨渠：下水铸坊与水声机关，寻找铁匠印记。
3. 盐白书库：盐晶档案塔，发现王冠契约。
4. 断弦温室：移动藤蔓与空中位移能力。
5. 黑曜巡礼道：旧王都外环，面对静音巡礼军。
6. 无声王冠：进入王冠核心，选择重铸钟声或切断封印。

## 本章NPC

- 织线学徒：操作教学与主线引子。
- 守门朝圣者：告诉玩家正门锁住，要找三把钟钥。
- 升台绘图师：解释地图图例和已探索房间。
- 后门司铃人：解释三钥匙拉杆、捷径和首领检查点。
- 回廊斥候：提示假灯、首领只有近战撞击、保留60能量回血。

## 地图引导字段

每个房间新增 `objective`、`guide`、`danger`、`next` 四个字段。地图面板会展示当前房间目标、路线提示、危险提示和下一步方向。
