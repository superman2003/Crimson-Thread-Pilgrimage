# 《绯线远征》6章魂游式箱庭地图 Layout

生成时间：2026-06-15 16:11:14.961

职责范围：Agent-Layout 只读建议文档，不改源码。本文用于关卡、剧情、美术继续拆分灰盒与资产需求。

核心目标：每章都突出“柳暗花明又一村”的箱庭体验：先让玩家在远景看到目标，再通过绕路抵达，从另一侧打开门或捷径，最后回到熟悉空间并形成空间顿悟。

## 0. 全局结构规则

### 0.1 节点标记

| 标记 | 含义 |
|---|---|
| S | 章节入口 |
| L | 灯碑/安全点 |
| V | 远景预告点，可看到 Boss 场、出口、稀有物或锁门背面 |
| D | 正面打不开的门、断桥、封印或机关锁 |
| K | 捷径门，必须从另一侧打开 |
| M | 机关、电梯、闸门、升降轴 |
| F | 单向落差、滑道、水流或坍塌段 |
| B | Boss 场 |
| X | 章节出口 |

### 0.2 每章必备回环

| 回环 | 体验作用 | 设计要求 |
|---|---|---|
| 主回环 | 从入口看见目标，但不能直达 | 入口 2 分钟内必须看到 Boss 场或出口轮廓 |
| 捷径回环 | 从背面打开门，缩短复跑 | Boss 前至少 1 扇门回到本章第一个 L |
| 机关回环 | 启动电梯/闸门后重新理解旧空间 | 机关开启后要让玩家从熟悉地点穿过 |
| 掉落回环 | 单向坠落后回到旧区域下层 | 掉落不惩罚到重跑过长，30-60 秒内回到主线 |

## 1. 翠锈钟庭

空间主题：锈蚀钟楼庭院，外环是花园和齿轮廊，内环是中央钟轴。第一章负责建立“看见中庭 Boss，却要绕过整个钟庭”的基本箱庭语法。

### 1.1 区域节点

| 节点 | 类型 | 内容与功能 |
|---|---|---|
| A0 断钟门桥 | S/L | 章节入口，第一个灯碑，远处可见中央大钟和 Boss 场屋顶 |
| A1 苔钟前庭 | V/D | 正面有铁栅门 K1，能看到中庭和回来的电梯井，但门从本侧打不开 |
| A2 翠锈齿廊 | 主路 | 基础跳跃、摆钟、齿轮平台教学，绕开 A1 正门 |
| A3 花钟竖井 | M/F | 第一处纵向空间，顶部看见隐藏物，底部单向落差回到 A2 下层 |
| A4 铜叶外庭 | 战斗环 | 外环庭院战斗，远景再次看到 Boss 钟面，强化目标感 |
| A5 断钟机房 | M/K | 启动齿轮电梯 M1，背面打开 K1 回到 A1 |
| A6 中庭司钟台 | L/B 前 | Boss 前灯碑，电梯直连 A1，复跑 8-12 秒 |
| A7 翠锈钟心 | B | Boss 场，圆形钟面平台 |
| A8 档案侧门 | X | Boss 后出口，通往“镜雨档案馆” |

### 1.2 空间层级

| 层级 | 区域 | 作用 |
|---|---|---|
| 上层 | 花钟竖井顶部、机房吊桥、司钟台 | 预告隐藏物和 Boss 前区域 |
| 中层 | 苔钟前庭、齿廊、铜叶外庭 | 玩家主要行进层 |
| 下层 | 花钟竖井底、齿轮排水沟 | 单向落差和回收路线 |
| 远景层 | 中央大钟、Boss 场屋顶 | 入口即展示终点 |

### 1.3 入口、出口与回环

| 类型 | 位置 | 说明 |
|---|---|---|
| 入口 | A0 断钟门桥 | 线性进入，安全观察 |
| 正面阻断 | A1 铁栅门 K1 | 玩家能看到中庭，但必须绕路 |
| 绕路抵达 | A2 -> A3 -> A4 -> A5 | 外环绕行一圈后到达门背面 |
| 捷径门 | A5 背面打开 K1 | 回到 A1，完成第一次“原来这里相通” |
| 机关捷径 | M1 齿轮电梯 | A6 直达 A1，服务 Boss 复跑 |
| 章节出口 | A8 档案侧门 | Boss 后开启 |

### 1.4 机关、单向落差、Boss 场

| 项目 | 位置 | 设计 |
|---|---|---|
| 摆钟机关 | A2/A4 | 周期摆动，可作为障碍和临时落脚点 |
| 齿轮电梯 M1 | A5 -> A6 -> A1 | 初见不可用，机房启动后贯通 |
| 单向落差 F1 | A3 顶部坍塌钟盘 | 掉到底层后发现 A2 下层熟悉空间 |
| Boss 场 | A7 翠锈钟心 | 位于地图中央偏上，入口 A0 和 A1 都能远眺；击败后钟面裂开露出 A8 |

## 2. 镜雨档案馆

空间主题：真实书库与镜中侧廊交错。第二章将“看见但不能直达”升级为“看见的是倒影，真实路径在背面”。

| B2 真实书库下层 | 主路 | 书架升降、墨雨障碍，建立基础探索 |
| B3 镜中侧廊 | M | 镜面机关切换真实/倒影通路 |

| B6 封存档案室 | L/B 前 | 背面打开 D1，形成 Boss 前复跑捷径 |
| B7 镜雨观测厅 | B | Boss 场，左右镜墙与双层平台 |
| B8 炉档转运口 | X | Boss 后通过档案传送槽去“余烬织炉” |

### 2.2 空间层级

| 层级 | 区域 | 作用 |
|---|---|---|
| 真实层 | B1/B2/B5 | 主探索和战斗 |

| 上层 | 倒影索桥、封存档案室上沿 | 远景预告和单向坠落 |
| 下层 | 墨雨积水库 | 失败回收、低风险补给 |

### 2.3 入口、出口与回环

| 类型 | 位置 | 说明 |
|---|---|---|
| 入口 | B0 雨廊入口 | 进入后立刻看到 B1 玻璃总目录 |
| 正面阻断 | B1 封存门 D1 | 只能从 B6 背面打开 |
| 绕路抵达 | B2 -> B3 -> B4 -> B5 -> B6 | 通过镜中侧廊绕到门后 |
| 捷径门 | B6 打开 D1 | Boss 前回到 B1，熟悉空间被重新解释 |
| 机关捷径 | M2 书柜电梯 | B5 下层直达 B1 上层 |
| 章节出口 | B8 炉档转运口 | Boss 后开启 |

### 2.4 机关、单向落差、Boss 场

| 项目 | 位置 | 设计 |
|---|---|---|
| 镜面切换 M1 | B3 | 拉动书签机关，真实平台和倒影平台交替实体化 |
| 书柜电梯 M2 | B5 | 初次从下往上走，启动后连接 B1/B6 |
| 单向落差 F1 | B4 索桥断点 | 落回 B2 下层，玩家从新角度认出早期书库 |

## 3. 余烬织炉

| C2 外炉环道 | 主路 | 沿炉体外侧绕行，热压、熔线、窄平台组合 |
| C3 绞盘竖井 | M/F | 链条电梯和单向坍塌格栅 |
| C4 煤渣下层 | 回收路 | 掉落后回到 C2 下方，可打开回梯 |
| C5 织线机房 | M/K | 关闭热浪门 D1，背面打开 K1 回到 C1 |
| C6 冷却回廊 | L/B 前 | Boss 前灯碑，连接 C1 的短路已开启 |
| C7 余烬织炉核 | B | Boss 场，三层炉架 |
| C8 冷凝水门 | X | Boss 后水汽冷凝，通往“盐月水渠” |

| 下层 | 煤渣下层、熔渣沟 | 单向坠落回收、隐藏补给 |

### 3.3 入口、出口与回环

## 4. 盐月水渠

空间主题：月盐闸门、上下双线水渠和水位切换。第四章通过“干渠远路安全、满潮近路危险”制造路线选择。

### 4.1 区域节点

| 节点 | 类型 | 内容与功能 |
|---|---|---|
| D0 冷凝水门入口 | S/L | 从织炉进入，低水位安全教学 |
| D1 月窗闸台 | V/D | 巨型月窗后能看到 Boss 闸心和出口水门 D1 |
| D2 干渠维护道 | 上层主路 | 较安全但绕远，适合初次探索 |
| D3 满潮下渠 | 下层副路 | 水流快、敌人密，可提前看到捷径背面 |
| D4 分流阀厅 | M/K | 切换水位，背面打开 D1 回到月窗闸台 |
| D5 逆流泵井 | M/F | 纵向泵井、电梯、水流滑道 |
| D6 月池净化间 | L/B 前 | Boss 前灯碑，水位稳定 |
| D7 盐月闸心 | B | Boss 场，三档水位变化 |
| D8 排空渡槽 | X | Boss 后水位下降，通往“骨枝空园” |

### 4.2 空间层级

| 层级 | 区域 | 作用 |
|---|---|---|
| 上层 | 干渠维护道、闸门走廊 | 安全绕路和远景观察 |
| 中层 | 月窗闸台、分流阀厅、净化间 | 主机关和 Boss 前回环 |
| 下层 | 满潮下渠、排水沟 | 高风险捷径和单向水流回收 |
| 纵向层 | 逆流泵井 | 连接上下水路，形成空间理解 |

### 4.3 入口、出口与回环

| 类型 | 位置 | 说明 |

| 入口 | D0 冷凝水门入口 | 进入后通过月窗看到 D7 闸心 |

| 捷径门 | D4 打开 D1 背闸 | 回到月窗闸台，玩家认出入口远景 |
| 机关捷径 | M1 泵井升降台 | D5 连接 D0/D4/D6 三处 |
| 章节出口 | D8 排空渡槽 | Boss 后水位下降露出出口 |

|---|---|---|
| 三档水位 M1 | D4/D7 | 低水位开下路，中水位开浮台，高水位开上层出口 |
| 逆流泵井 M2 | D5 | 水流向上喷发，玩家要借水流和平台上升 |
| 单向水滑 F1 | D3 末端泄洪道 | 被水冲回 D0 下方，从熟悉入口下层出来 |
| Boss 场 | D7 盐月闸心 | 入口 D1 可见，实际从 D6 进入；击败后排空水渠，D1 变成可通行旧门 |

## 5. 骨枝空园

| E2 根桥下层 | 主路 | 低处横向探索，学习风剪和脆骨平台 |
| E3 空中温室 | V/F | 高处看见 E0 和 E1，单向坠落回根部 |

| E7 骨枝空巢 | B | Boss 场，悬空圆形巢台 |

## 7. 整体验收标准

| 验收项 | 标准 |
|---|---|
| 远景预告 | 每章入口 2 分钟内能看到 Boss 场、出口或关键锁门背面 |
| 绕路抵达 | 每章至少 1 条主绕路，经过 4-6 个节点后回到预告点背面 |
| 背面开门 | 每章至少 1 个 K 捷径门从另一侧打开 |
| 熟悉空间回归 | 每章至少 1 次从新角度回到入口附近或早期主路 |
| Boss 复跑 | Boss 前灯碑到 Boss 门控制在 8-15 秒 |
| 单向落差 | 每章至少 1 个 F，落下后不超过 60 秒回主线 |
| 终章收束 | 第 6 章把前 5 章机制各复现一次，但每段只做短考，不抢最终 Boss 节奏 |
