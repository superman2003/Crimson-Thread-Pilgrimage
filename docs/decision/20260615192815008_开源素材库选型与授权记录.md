# 开源素材库选型与授权记录

## 决策

本轮运行时素材优先切换到 Kenney `Platformer Art Deluxe`，授权为 Creative Commons CC0。

## 原因

- CC0，允许个人和商业项目使用。
- 本地包内 `license.txt` 明确说明 credit 非强制。
- 2D 平台跳跃素材完整，包含背景、Tile、敌人、道具、HUD、角色帧。
- 可以同名覆盖当前 Godot PNG，不改变碰撞、关卡、AI 和输入逻辑。

## 来源

- 素材库：Kenney Platformer Art Deluxe
- 来源链接：https://kenney.nl/assets/platformer-art-deluxe
- 本地源包：`artifacts/source_assets/kenney/kenney_platformer-art-deluxe.zip`
- 解压目录：`artifacts/source_assets/kenney/platformer-art-deluxe`
- 本地授权文件：`artifacts/source_assets/kenney/platformer-art-deluxe/license.txt`
- 运行时素材清单：`godot/assets/sprites/open_asset_manifest.json`

## 未采用项

- itch.io CC0 Assets：资源质量和授权需要逐页确认，适合下一轮按具体美术方向筛选。
- OpenGameArt：授权混合，适合单件素材引入，但需要逐项记录 CC0/CC-BY/CC-BY-SA/GPL。
- Game-icons.net：CC BY 3.0，需要署名；本轮暂不引入，避免增加 attribution 复杂度。
- Freesound：授权混合，本轮未做音效替换。
- Poly Haven / ambientCG / Quaternius：更适合 2.5D、3D 或材质路线，本轮 Godot 2D Demo 暂不使用。

## 约束

- 只替换 PNG 和动画 manifest，不修改碰撞体。
- `SPRITES` 字典路径保持不变，避免改动运行时代码。
- 资源验证必须覆盖全部 24 个运行时 demo 素材和 14 套主角动画。
