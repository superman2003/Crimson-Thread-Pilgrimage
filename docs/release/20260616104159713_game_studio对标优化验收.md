# Game Studio 对标优化验收

## 结果

- 当前可运行版本仍是 Web Canvas / Node 静态服务，不是 Godot 引擎运行。
- 已根据 `docs/spec/20260615235253725_空洞骑士丝之歌对标优化规格.md` 完成 P0/P1 优化。
- 对标目标限定为动作平台质量标准，不在运行时代码和文案中使用 Hollow Knight / Silksong / Team Cherry 等受保护 IP 名称。

## 改动

- `src/main.js`
  - 增加 `PLAYER_TUNING`、跳跃缓冲、土狼时间、短按跳高控制、落地反馈。
  - 增加攻击前冲、命中停顿、普通敌击退、敌人受击与 Boss 受击事件。
  - 增加 shooter 预警线、charger windup 预警。
  - 增加真实钥匙收集、捷径开启、捷径桥碰撞、目标方向箭头。
  - 扩展 `window.__crimsonThreadQA`：`collectAllKeys()`、`openShortcut()`、`strikeFirstEnemy()`、`snapshot()` 字段。
- `src/gameData.js`
  - 修复 6 章中文乱码。
  - 每章补齐 `flow`、3 把 `keys` 和 `shortcut` 配置。
- `index.html`
  - 修复标题、HUD、按钮和触控按钮乱码。
- `tests/*`
  - 增加中文乱码哨兵、运行时 IP 禁词哨兵、钥匙/捷径/命中停顿机制断言。
  - 浏览器测试验证主角帧、QA hook、新机制字段和桌面/手机截图。
