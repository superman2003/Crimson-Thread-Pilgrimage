# CH01 回爬、敌人出生、Boss 体积动作与章节材质修复

## 结果

- 修复 CH01 外庭掉落后回不去的问题：拆短压住回爬通道的 `shortcut_upper`，并加入净空校验。
- 敌人出生改为硬性清空扫描：候选点不清空时会扫描全章节可用平台，不再直接退回墙内原坐标。
- Boss 体积改为数据驱动：六章 Boss 均补 `body_size`、`hurtbox_size`、`hurtbox_offset`，运行时同步校验。
- Boss 攻击补关键帧姿态：六章 Boss 攻击均带蓄力、出手、恢复的缩放和偏移参数。
- CH03-CH06 不再复用 CH01 平台材质名：分别切到盐白书库、断弦温室、黑曜巡礼道、无声王冠主题材质。
