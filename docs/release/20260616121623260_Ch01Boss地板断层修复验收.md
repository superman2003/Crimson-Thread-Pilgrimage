# Ch01 Boss地板断层修复验收

时间：2026-06-16 12:16:23.260

## 问题

第一章 Boss 区扩图后发生数据错位：`boss_chamber` 房间范围仍在旧区间，Boss 门、Boss 本体和 Boss 地板被移到后续扩展房间附近；同时 `backdoor_room` 到 `boss_runback` 中间存在大段无平台区域，导致玩家到 Boss 前后看起来没有地板或无法稳定抵达。

## 修复

- 重排 Ch01 后半段房间顺序：`runback -> ch01_expanded_1 -> ch01_expanded_2 -> boss_chamber -> ch01_expanded_3`。
- 将 Ch01 Boss 检查点、Boss 门、Boss 本体、Boss 地板、章节出口重新对齐到同一条可走路线。
- 同步修正 Ch02-Ch06 同模板生成的后段断层，避免后续章节重复出现同类问题。
- 在 `all_chapters_demo_validation.gd` 增加硬验证：
  - Boss 检查点脚下必须有平台。
  - Boss 门脚下必须有平台。
  - Boss 出生点脚下必须有平台。
  - 章节出口脚下必须有平台。
  - Boss 门和 Boss 出生点必须落在 Boss 房间范围内。
  - 后段路线不能出现大断层。
