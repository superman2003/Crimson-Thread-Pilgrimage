# 敌我 NPC 素材分流修正

## 结果

- 敌人和 Boss 不再使用与主角同源的 Abyss/Itch 虫系素材。
- 敌人统一切换为 `Ansimuz GothicVania Cemetery` 开源免费素材，来源记录见 `godot/assets/sprites/gothicvania/asset_manifest.json`。
- 原同族素材仅保留给 NPC，并额外加绿色友方交互标识，避免误判为敌人。
- 已删除本轮临时生成的 `distinct` 自绘素材目录和生成脚本。
