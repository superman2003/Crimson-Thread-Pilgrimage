# 六章可玩闭环与开源免费素材策略交付

## 结果

已完成 CH01-CH06 六章最小可玩闭环。所有章节都能通过同一个 `Main.tscn` 注入不同 runtime config，完成 NPC 对话、章节敌人生成、三钥匙捷径、Boss 门、Boss 击杀与章节奖励发放。

## 素材策略

- 主角、敌人、Boss 使用项目已登记的 GothicVania / Kenney / Mossy Cavern / 本地 OGG 等免费、开源或可商用友好素材链路。
- 旧同族虫系素材只作为友方 NPC 使用，不再给敌人或 Boss 使用。
- 每个 NPC 保留 `FriendlyTalkBadge` 与 `FriendlyHalo`，保证敌我辨识度。
- 未新增 AI 生成图片素材，也未引用 `enemies/distinct` 或付费必需素材。
