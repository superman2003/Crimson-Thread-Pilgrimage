# 六章地图素材文件级去复用

## 结果

- 在 Godot 运行时接入三套额外开源地图素材：
  - GothicVania Cemetery by Ansimuz，Public Domain。
  - Kenney Platformer Art Deluxe，CC0。
  - Godot Platformer 2D 背景素材，MIT。
- CH03-CH06 不再只靠材质名和 tint 区分：
  - CH03 使用 GothicVania Cemetery `tileset.png / background.png / mountains.png / graveyard.png`。
  - CH04 使用 Kenney Deluxe `grass / grassHalf / dirt` 平台文件与 `bg.png`。
  - CH05 使用 Kenney Deluxe `stone / castle` 平台文件与 `bg_castle.png`。
  - CH06 使用 Kenney Deluxe `snow / snowHalf` 平台文件与 Godot Platformer 2D `sky.png / mountains.png`。
- 全章节验证升级为“平台图片路径 + 背景图片路径”均存在且跨章节不复用。
- 额外拆掉 CH05/CH06 与前章重复的敌人视觉签名，并升级验证为“敌人视觉集合跨章节不复用”。
