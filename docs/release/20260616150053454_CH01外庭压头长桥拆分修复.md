# CH01 外庭压头长桥拆分修复

## 结果

已修复第一章外庭回爬路线被上方长桥顶住的问题。此前 `shortcut_upper` 横跨 x=1180 到 x=3320，实际形成实心天花板，导致玩家从下层跳回主路线时被挡住。

## 改动

- `godot/data/demo_ch01_moss_bell_court.json`
  - 将 `shortcut_upper` 从宽度 `2140` 缩短为 `500`，保留左侧短桥表现，打开中段回爬井空间。

- `godot/scripts/demo_validation.gd`
  - 新增 `outer court return climb shaft has headroom` 断言。
  - 新增 `_climb_corridor_clear()`，检查回爬路径上方没有非路径平台形成天花板。
