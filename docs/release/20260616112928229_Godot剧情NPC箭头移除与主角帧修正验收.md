# Godot剧情NPC箭头移除与主角帧修正验收

时间：2026-06-16 11:29:28

## 结果

已完成本轮 Godot 版本整改：移除场景内箭头引导，NPC 改为带身份、动机、帮助信息和多句剧情对话，六章剧情蓝图已落盘，主角 idle/run 帧高度和脚底基线已统一。

## 改动范围

- `godot/scripts/main_level.gd`
  - 删除箭头式 guide marker 生成逻辑。
  - 存档点改为非箭头标记。
  - 新增 `DialogueOverlay` 对话面板。
  - NPC 交互从简短 toast 升级为姓名、身份、剧情台词、帮助提示。
  - 运行状态增加 `guide_marker_total` 和 `dialogue_overlay_ready` 供验证脚本检查。
- `godot/data/demo_ch*.json`
  - 六章均补齐 `world_story`。
  - 六章 `guides` 均置空。
  - 25 个 NPC 均补齐 `identity`、`motive`、`help` 和不少于 3 句中文剧情对话。
- `godot/assets/sprites/generated/hero_v2/run_00.png` 至 `run_03.png`
  - 清理残留格线/边角标记。
  - 统一 run 与 idle 的可见高度和脚底基线，降低站立/行走时的缩放跳变。
- `godot/scripts/story_guidance_validation.gd`
  - 新增剧情、NPC、无箭头、对话面板专项验证。
- `godot/scripts/chinese_story_validation.gd`
  - 更新为验证 NPC 对话面板和中文剧情信息。
- `godot/scripts/art_direction_validation.gd`
  - 增加主角 idle/run 帧高度和脚底基线一致性检查。
- `docs/spec/20260616111921178_绯线巡礼剧情与NPC蓝图.md`
  - 新增完整剧情策划文档，包含世界观、主角、六章走向、NPC身份和结局结构。
