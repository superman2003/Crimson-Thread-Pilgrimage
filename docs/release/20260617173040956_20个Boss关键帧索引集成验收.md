# 20 个 Boss 关键帧索引集成验收

结果：20 个 Boss 的攻击关键帧已全部纳入 artifacts 与 Godot 索引，预览页和总览图已重建。

## 范围

- 素材目录：`artifacts/boss_keyframes`
- Godot 目录：`godot/assets/sprites/bosses`
- 关键帧规格：每个 Boss 6 帧 attack，artifacts 帧为 384x384，Godot 帧为 256x256。
- 本次处理方式：复用已有 boss_01 到 boss_20 PNG 帧与 strip，不重画、不覆盖图片本体。
