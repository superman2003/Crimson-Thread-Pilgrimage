# 主角缩放与可见文本策略修正

## 结果
- 主角 `AnimatedSprite2D` 从 `scale = Vector2(1.25, 1.25)` 降到 `Vector2(0.92, 0.92)`，站在 Mossy Cavern 台阶上不再过大。
- 关卡内路牌中文、NPC 头顶中文、地图中文说明、HUD 中文、背包中文全部从可见层移除。
- 仅保留 NPC 进入交互范围时的提示 `按F交谈`，符合当前交互按钮要求。

## 改动
- `godot/scenes/Player.tscn`：缩小主角精灵并调整垂直偏移。
- `godot/scripts/main_level.gd`：新增英文房间、地图目标、道具、NPC 名称与对话显示表；停止渲染世界中文标签；路牌只保留图标。
- `godot/scripts/chinese_story_validation.gd`：改为可见文本策略验证，允许唯一中文交互提示。
- `godot/scripts/inventory_map_validation.gd`：更新背包和地图英文验收。
- `godot/scripts/art_direction_validation.gd`：加入主角缩放断言。

- JSON 数据中仍保留中文剧情底稿，但当前运行时不直接显示这些字段；后续如果新增 UI 读取配置字段，需要继续走显示映射层。
