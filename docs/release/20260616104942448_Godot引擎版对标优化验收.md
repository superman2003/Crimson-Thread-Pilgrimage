# Godot引擎版对标优化验收

## 结果

Godot 版已经作为主交付路线完成验证；当前不是网页版目标。

## 改动摘要

- `godot/scripts/player_controller.gd`
  - 玩家素材切到 `res://assets/sprites/player/generated_hero_v2/manifest.json`
  - 增加 image2 主角 manifest 解析。
  - 优化移动、跳跃缓存、土狼时间、短跳、冲刺、攻击前冲和落地动画。
- `godot/scenes/Player.tscn`
  - 调整主角显示缩放，避免 192px 新帧在关卡里过大。
- `godot/scripts/art_direction_validation.gd`
  - 改为验收 generated hero v2、苔藓平台和地图纹理。
- `godot/scripts/inventory_map_validation.gd`
  - 改为验收中文背包道具和中文地图房间名。
- `godot/scripts/chinese_story_validation.gd`
  - 改为正向验收中文剧情、NPC 名、背包和地图文本。
