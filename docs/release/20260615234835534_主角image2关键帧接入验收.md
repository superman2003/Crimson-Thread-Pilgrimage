# 主角 image2 关键帧接入验收

## 结果

- 主角已从旧 GothicVania 单帧切换为 image2 生成的 `generated_hero_v2` 关键帧素材。
- 碰撞盒保持 `34x54`，视觉渲染独立放大到 `128x128`，不改变平台跳跃手感。
