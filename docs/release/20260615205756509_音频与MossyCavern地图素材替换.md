# 音频与 Mossy Cavern 地图素材替换

## 范围
- 地图素材改为 Maaot 的 Mossy Cavern，导入位置：`godot/assets/third_party/mossy_cavern/`。
- 保留游戏内使用，不把 Mossy Cavern 原始素材重新打包成独立素材包分发。
- 音频补齐背景音乐、Boss 音乐、攻击、跳跃、冲刺、回血、受击、拾取、互动等 13 个音效资源。
- 增加回血绿色丝线/圆环特效，增加命中火花特效。
- 地图面板加入 Mossy Cavern 背景图，主场景平台、背景、藤蔓装饰改用正式素材。

## 来源与授权
- Mossy Cavern：`https://maaot.itch.io/mossy-cavern`。页面授权允许免费/商业项目使用与修改，不强制署名；限制是不能将素材重新打包、再发布或转售。
- Bread Adventure 音频：`https://github.com/wheafun/bread-adventure`，Apache-2.0。

## 风险
- Mossy Cavern 可以用于游戏内成品，但不要把原始素材或裁剪后的素材单独作为素材包发布。
- Godot headless 音频专项脚本会在退出时打印一次对象泄漏警告，但验证返回码为 0；普通地图、战斗、背包、Demo 验证脚本退出干净。
