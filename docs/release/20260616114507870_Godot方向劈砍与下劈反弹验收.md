# Godot方向劈砍与下劈反弹验收

时间：2026-06-16 11:45:07

## 结果

已完成类空洞骑士/丝之歌方向劈砍整改：普通攻击支持左、右、上、下四个方向；下劈命中敌人会向上反弹；玩家挥刀期间擦到敌人身体不再被接触伤害卡死，但敌人的主动招式仍然可以造成伤害。

## 改动范围

- `godot/scripts/player_controller.gd`
  - 新增 `current_attack_direction`，攻击瞬间读取方向输入。
  - 新增 `aim_up`、`aim_down` 输入动作。
  - Space 保留为跳跃，W/上方向改为上劈瞄准，S/下方向改为下劈瞄准。
  - `_apply_attack_hits()` 和 `_sync_attack_hitbox()` 改为按方向生成攻击矩形。
  - 下劈命中时执行 `downslash_bounce_velocity`，让玩家向上弹起。
  - 新增 `can_ignore_enemy_contact()`，用于攻击/反弹短时间内忽略敌人身体接触伤害。
  - 普通劈砍和技能劈砍 VFX 根据上下方向调整显示。
- `godot/scripts/enemy_actor.gd`
  - 身体接触伤害调用前检查玩家是否处于攻击保护状态。
  - 保留敌人主动招式 `_try_damage_player()` 的伤害，不把攻击做成全程无敌。
- `godot/scripts/combat_validation.gd`
  - 增加左劈、上劈、下劈、下劈反弹、攻击期身体接触豁免、保护结束后接触伤害恢复等断言。
- `godot/scripts/main_level.gd`
  - HUD 控制提示更新为 `A/D Move W/S Aim Space Jump Shift Dash LMB/J Slash`。

## 操作手感

- 左/右劈：按住 A/D 或左右方向后按 LMB/J。
- 上劈：按住 W/上方向后按 LMB/J。
- 下劈：空中或站位高于敌人时按住 S/下方向后按 LMB/J，命中后会向上弹。
- 攻击期间与敌人身体重叠不会立刻掉血，便于从怪物身上越过去；敌人蓄力/挥击的攻击框仍会打到玩家。
