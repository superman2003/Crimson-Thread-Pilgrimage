# CH02铸雨渠最小可玩闭环

## 结果

已完成第二章“铸雨渠”的最小可玩闭环：主场景可注入章节配置，CH02 可从入口跑到收集钥匙、拉水轮捷径、打开 Boss 门、击败沉锤铁匠并获得铁匠誓牌。

## 改动

- `godot/scripts/main_level.gd`
  - 保留 CH01 默认入口，新增 `config_path` 注入能力，后续章节可复用同一个 Main 场景。
  - NPC 名称、台词、地图房间名、地图目标说明、Boss 唤醒文案、通关奖励改为优先读取章节 JSON。
  - 新增 CH02 敌人 kind 映射，敌人只引用 `godot/assets/sprites/gothicvania/demo/*` 免费开放素材。
  - 新增 CH02 NPC role 映射，旧同族素材只保留为友方 NPC，并继续显示友方标识。
- `godot/data/demo_ch02_rain_foundry_canal.json`
  - 新增 CH02 可运行配置：7 个房间、4 个 NPC、5 个普通敌人、3 个泵井钥匙、6 个陷阱、1 个 Boss。
  - 明确素材策略：敌人使用 GothicVania 开放素材，NPC 使用友方专用 Itch 旧同族素材，UI/VFX 使用 Kenney CC0 与程序化特效。
- `godot/data/demo_ch01_moss_bell_court.json`
  - 补齐第一章 Boss 名称、掉落和中文通关文案，避免通用兜底文案露出。
- `godot/data/campaign_chapters.json`
  - 第二章新增 `runtime_config_id: demo_ch02_rain_foundry_canal`，运行时可从 demo config 映射回“铸雨渠”。
- `godot/scripts/ch02_demo_validation.gd`
  - 新增 CH02 运行时验证：配置加载、章节映射、NPC 对话、敌人 kind、钥匙、拉杆、Boss 门、Boss 击杀、奖励发放。
