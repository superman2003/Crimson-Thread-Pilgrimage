# Godot陷阱存档与刷怪修复验收

## 结果

本轮修复已落到 Godot 工程，覆盖主角站立抽搐、切图残留、陷阱传送、怪物刷墙、存档点复活五个问题。

## 改动摘要

- 主角 idle 改为单帧，不再站立循环 4 帧。
- 清理 `generated_hero_v2/frames` 中的底部编号、切图竖线和孤立脏点。
- 陷阱触发只扣血，不再调用 `respawn_at(active_spawn)`。
- 坠落出地图按死亡处理，死亡结束后回当前激活存档点。
- 捷径开启不再暗改复活点，复活点只由玩家按 F 激活存档点决定。
- 新增 `save_points` 数据，每章 2 个，至少 1 个隐藏点。
- 怪物生成增加平台吸附校正，排除 wall/gate/locked 等不可走平台。
- 验证脚本补充陷阱不传送、存档点复活、怪物平台归属、idle 单帧断言。
