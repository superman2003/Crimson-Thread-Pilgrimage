# Boss 全量多招式 VFX 交付记录

## 范围

- 覆盖 Boss：`boss_01` 到 `boss_20`
- 特攻数量：60
- 每个特攻帧数：6
- Artifact 帧输出：`artifacts/boss_keyframes/<boss>/actions/<attack_id>/attack_00.png` 到 `attack_05.png`
- Godot 帧输出：`godot/assets/sprites/bosses/<boss>/frames/<attack_id>/attack_00.png` 到 `attack_05.png`
- 共享索引：
  - `artifacts/boss_keyframes/boss_special_attack_keyframes_index.json`
  - `godot/assets/sprites/bosses/boss_special_attack_keyframes_index.json`
- 预览材料：
  - `artifacts/boss_keyframes/contact_sheets/boss_special_attack_keyframes_contact_sheet.png`

## 风险与回滚

- 风险：生成式素材存在风格细节波动，后续可按单个 Boss 或单个 attack 局部替换，不影响索引结构。
- 回滚：可删除对应 `actions/<attack_id>`、`frames/<attack_id>` 和 manifest 中同名 attack 后，重新运行导入脚本重建索引。
