# Itch 敌人动画素材替换记录

## 范围
- 普通虫系敌人和 Boss 改用 Abyss Bug Sprite Pack。
- 孢子类敌人改用 Forest Monsters 2D Pixel Art 免费版 Mushroom。
- 新增敌人动画 manifest，包含 `idle`、`walk`、`attack`、`hurt`、`death`。
- 保留旧 PNG 兜底能力：`enemy_actor.gd` 遇到 manifest 播动画，遇到普通 PNG 仍显示静态贴图。
- 不改关卡 JSON、不改敌人 AI 数值、不新增远程法术。

## 来源
- Abyss Bug Sprite Pack: `https://arhimed122.itch.io/abyss-bug-sprite-pack`
- Forest Monsters 2D Pixel Art: `https://monopixelart.itch.io/forest-monsters-pixel-art`

## 实际映射
- `moss_larva` -> Abyss mantis duelist
- `bronze_moth` -> Abyss moth wisp
- `spore_bellmaker` -> Forest Monsters FREE Mushroom
- `gear_sentinel` -> Abyss armored beetle guard
- `rust_crown_guardian` -> Abyss stag beetle monarch boss

## 授权注意
- Abyss 页面说明允许个人/商业项目使用和修改，禁止作为独立素材包转卖或再分发。
- Forest 页面说明免费版可用于项目，但不要转卖、再分发或作为独立素材修改发布。
- 本轮只使用 `Forest_Monsters_FREE.zip` 中的 Mushroom，不引用 Premium 的 Slime 或 Bush Monster。

## 已知事项
- `audio_visual_validation.gd` 在 headless 音频专项退出时仍会打印一次 ObjectDB 泄漏警告，但脚本返回码为 0，所有验证项通过。
- Abyss 源图是白底精灵表，运行时帧通过自动扣底与底部锚点规范化生成；后续如购买更完整透明帧包，可直接替换 manifest 里的帧文件。
