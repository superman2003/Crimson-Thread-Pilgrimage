# 类丝之歌地图点位策划素材调研

## 结论

有可用素材，但不要找“照搬丝之歌地图”的素材；更靠谱的是三类组合：

1. 地图系统参考：用 Godot Metroidvania System 学房间图谱、地图显示、收藏物/对象追踪。
2. 点位策划模板：自己建立每个房间的敌人、陷阱、奖励、能力门、回环、NPC、检查点表。
3. 美术素材库：用 itch.io / Kenney / OpenGameArt 的 tileset、敌人、UI、图标填充具体房间。

## 布怪原则

- 一个房间只教一个主要问题：跳跃、冲刺、怪物前摇、地形限制、回环、奖励风险。
- 敌人按组合放，不按数量堆：近战撞击怪 + 地形压迫 + 可撤退空间。
- 怪物间距至少给玩家一次反应和恢复：普通房 240-420px，精英房 480px 以上。
- 同屏怪物控制在 1-3 只；Boss 前回廊更少，但单体伤害和地形压力更高。
- 每章至少有：教学房、技能门房、回环房、奖励房、陷阱房、NPC 房、Boss 前检查房、Boss 房。
