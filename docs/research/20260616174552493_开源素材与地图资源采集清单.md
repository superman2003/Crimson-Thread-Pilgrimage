# 开源素材与地图资源采集清单

时间：2026-06-16 17:45:52.493
项目：绯线巡礼 / Godot 4.6 / 2D 动作平台类银河城
目标：沉淀可用于本项目的素材来源、地图设计工具、授权边界和导入流程。

## 1. 总结

本项目可以继续扩大素材来源，但要把“免费”“可商用”“开源可再分发”分开处理。

优先策略：

1. 公共仓库优先使用 `CC0 / Public Domain / MIT / Apache-2.0 / OFL`。
2. `CC-BY / OGA-BY` 可以使用，但必须保留作者、来源、授权和修改说明。
3. `CC-BY-SA / GPL` 不建议混入当前公开仓库的主美术库，除非明确接受派生素材同协议共享。
4. CraftPix、GameDev Market、Sonniss、Pixabay、很多 itch.io 付费包属于“可商用但不一定开源”，适合本地授权目录或闭源发行包，不适合公开再分发原始 PNG/PSD/ASE。
5. Pixel Joint 主要做风格参考和找画师，不当素材库。
6. The Spriters Resource 主要是商业游戏媒体归档，不用于运行时素材。

| The Spriters Resource | 禁用为运行素材 | 可学习拆图结构和动画组织，不用于项目素材 |
| GameDev Market | 商业素材市场 | 适合采购 UI、背景、音效；不公开再分发原始素材 |

## 5. 地图设计工具链

| 工具 | 类型 | 用途 | 推荐 |
|---|---|---|---|
| Godot TileMapLayer / TileSet | Godot 原生地图工具 | 最终运行地图、碰撞、自定义数据、多层 tile | P0 |
| Godot Parallax2D | 视差背景节点 | 远景、中景、近景滚动 | P0 |
| LDtk | 2D 关卡编辑器 | 房间网格、世界图、门连接、实体点位 | P0 |
| Godot LDtk Importer | Godot 导入插件 | `.ldtk` 导入 Godot Scene | P0 |
| Tiled Map Editor | 通用 tilemap 编辑器 | `.tmx/.tmj` 传统地图流程 | P1 |
| Ogmo Editor 3 | 2D 关卡编辑器 | 简洁 JSON 输出；适合自写导入适配 | P2 |

推荐路线：LDtk 负责关卡蓝图和房间连接，Godot 内部使用 `TileMapLayer` 拆成 `Terrain / DecorBack / DecorFront / Hazard / OneWay / Secret`，背景用 `Parallax2D` 做 3 到 5 层。

## 7. 推荐项目目录规范

公开可再分发素材：

```text
godot/assets/third_party/cc0/<source_or_pack>/
godot/assets/third_party/cc_by/<source_or_pack>/
godot/assets/third_party/mit/<source_or_pack>/
```

本地授权或不可再分发素材：

```text
artifacts/third_party/<vendor_or_pack>/
godot/assets/licensed/<vendor_or_pack>/
```

`godot/assets/licensed` 如果未来使用，应加入 `.gitignore` 或只在私有仓库存放。公开仓库只提交引用说明、下载指引和授权记录。

## 9. 决策记录

| 决策 | 说明 |
|---|---|
| 首批地图素材优先 CC0 | 避免“可商用但不能公开再分发”的历史问题 |
| LDtk 优先于 Tiled | 房间、实体点位、世界网格更贴合类银河城；Tiled 作为备选 |
| Pixel Joint 不进素材库 | 只做风格参考，避免未授权作品进入项目 |
| The Spriters Resource 禁用为运行素材 | 归档商业游戏媒体，版权风险高 |
| CraftPix/GameDevMarket 归入 licensed | 可商用不等于可开源分发，必须和公开仓库隔离 |
