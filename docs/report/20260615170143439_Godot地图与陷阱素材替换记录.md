# Godot 地图与陷阱素材替换记录

时间：2026-06-15 17:01:43.439  
范围：第 1 章试玩 demo 的地图背景、平台材质、陷阱素材替换。

## 结果

已把上一版偏灰盒的地图/陷阱视觉替换为具象素材：

- 地图背景：使用 D 盘 v2 环境面板生成 `3840x720` 背景。
- 平台材质：新增苔石、铜桥、Boss 石台 3 类平台贴图。
- 陷阱素材：新增 6 张独立陷阱贴图，对应 6 个陷阱配置。
- Godot 关卡脚本：平台和陷阱现在按 JSON 的 `material` / `kind` 字段加载素材。

## 配置接入

关卡配置：`godot/data/demo_ch01_moss_bell_court.json`

- `platforms[].material`：`moss_stone` / `bronze_bridge` / `boss_stone`
- `hazards[].kind`：`fake_moss_floor` / `bell_gap` / `spore_chest` / `falling_clapper` / `shortcut_revenge` / `false_lamp`

关卡脚本：`godot/scripts/main_level.gd`

- 背景加载 `map_background_ch01.png`
- 平台加载 material 对应贴图
- 陷阱加载 kind 对应贴图
