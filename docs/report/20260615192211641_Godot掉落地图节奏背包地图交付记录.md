# Godot 掉落地图节奏背包地图交付记录

## 结果

本轮已修复玩家掉出地图后的灰屏体验，扩大第一章地图并降低怪物密度，移除怪物远程/法术方向，新增背包系统与地图系统，并通过本地 Godot 与脚本验证。

## 改动文件

- `godot/scripts/main_level.gd`
  - 新增背包覆盖层、地图覆盖层、房间访问记录、地图绘制。
  - 钥匙拾取进入背包。
  - 普通怪与 Boss 死亡掉落货币和材料。
  - HUD 增加 `Tab Bag` 与 `M Map` 控制提示。
- `godot/data/demo_ch01_moss_bell_court.json`
  - 世界宽度保持 `6400`。
  - 普通怪保持 5 个，节奏更疏。
  - 新增 `inventory` 与 `map_rooms` 配置。
  - 新增 `Moss Chitin` 材料。
- `godot/scripts/inventory_map_validation.gd`
  - 新增背包、地图、访问房间、掉落入包、Esc 关闭覆盖层验证。
