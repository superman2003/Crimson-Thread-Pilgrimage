# Godot 开源素材替换交付记录

## 结果

已将当前 Godot Demo 的运行时素材切换为 Kenney `Platformer Art Deluxe` CC0 素材路线，并保留本地源包、生成脚本、授权清单和验证脚本。

## 改动

- `tools/build_open_asset_replacements.py`
  - 从 Kenney 源素材生成 Godot 运行时 PNG。
  - 替换背景层、平台、敌人、Boss、陷阱、钥匙、拉杆、Boss 门。
  - 生成 Kenney p3 主角 14 套动画帧与动画 manifest。
- `tools/validate_open_asset_replacement.py`
  - 验证 24 个 demo 运行时素材均存在、为 RGBA、背景尺寸为 `8192x720`。
  - 验证主角 14 套动画帧均为 `128x128`。
  - 验证 manifest 来源为 Kenney CC0。
- `godot/assets/sprites/open_asset_manifest.json`
  - 记录素材来源、授权、下载地址、本地源目录和运行时目标。
- `godot/assets/sprites/demo/*.png`
  - 已按同名文件替换为 Kenney 风格素材。
- `godot/assets/sprites/lumen/*`
  - 主角动画已切换为 Kenney p3 帧，保留原运行时路径结构。

## 风险

- Kenney 风格更干净、授权更稳，但偏原型/卡通，不是最终黑暗类银河城风格。
- `tools/build_godot_demo_assets.py` 仍会生成旧临时素材；后续应改为调用 `build_open_asset_replacements.py` 或弃用旧脚本，避免误覆盖。
- 当前没有引入 Game-icons.net，因此不需要额外署名；后续引入 CC-BY 素材时必须维护 attribution。
