# 类丝之歌地图设计规则接入生成器报告

## 结果

已把“丝之歌式类银河城地图设计原则”转成原创规则，并接入现有 `blueprint -> Godot demo JSON` 生成链路。

本轮没有复刻商业地图布局，只抽象通用关卡结构：

- 主线清晰推进。
- 支线有奖励和回流。
- 纵向井承担记忆点。
- 捷径从背面开启并节省路径。
- 能力获得后有教学房和回访门。
- Boss 前有短跑道和存档点。

## 当前状态

当前生成器已经从“可达性灰盒”升级到“带关卡设计契约的灰盒”。

还不是最终商业级地图，原因是：

- 房间内部机关仍是模板化平台。
- NPC、任务、隐藏奖励还没有完整事件链。
- 美术主题还停留在材质/颜色/布局层，不是 TileMap 级场景块。
- 玩家实机跑图节奏还需要 playtest。
